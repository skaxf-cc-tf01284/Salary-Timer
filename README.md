# Salary Timer

Ứng dụng đếm ngược giờ về và ngày nhận lương (UTC+7), build bằng React + Vite.

## Scripts

- `npm run dev`: chạy local
- `npm run build`: build production
- `npm run preview`: chạy bản build local

## Deploy GitHub Pages

Project đã có workflow tại `.github/workflows/deploy-pages.yml`.
Chỉ cần push lên nhánh `main`, GitHub Actions sẽ tự build và deploy.
